ICON PLACEHOLDER
================

To complete the extension, you need to add an Icon.png file in this directory.

Requirements:
- Size: 128x128 pixels (minimum)
- Format: PNG with transparency
- File name: Icon.png

You can:
1. Create your own icon using any image editor
2. Use the robot icon from the VS Code extension (local-llm-chat-vscode-main/resources/robot_icon.png)
3. Copy any suitable PNG icon and resize it to 128x128

Quick steps:
1. Find or create a 128x128 PNG icon
2. Save it as: Resources/Icon.png
3. Rebuild the project

Without an icon, the build may fail or show a default icon.
